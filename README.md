## 🔎 About rmp

rmp is a proof-of-concept app that shows how you can build multi-platform applications using a clear architecture guide. This app uses Rust and supports many platforms, including desktop and mobile.

The project includes several parts:

- A detailed guide on building apps using the "RMP Architecture Bible."
- A command-line tool called `rmp` that helps set up new projects.
- An example chat app to demonstrate the technology in action.

You do not need to understand software development to use rmp, but this README focuses on helping you get the software running on Windows.
